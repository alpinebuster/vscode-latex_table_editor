# vscode-latex_table_editor

An Open Source WYSIWYG table editor that exports to multiples languages including LaTeX, ConTeXt, Plain TeX, CSV, HTML, BBCode, Eplain, PreTeXt, MediaWiki, Markdown and WML.

## Features
### Formats supported

 - LaTeX
 - ConTeXt
 - Plain TeX
 - PreTeXt
 - Markdown
 - HTML
 - CSV
 - BBCode
 - Textile
 - Eplain
 - WML
 - MediaWiki
 - Text (coming soon)
